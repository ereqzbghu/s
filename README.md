# simpleC

## 介绍
simpleC是Android上的C/C++语言开发工具，可以编辑并编译C/C++代码，并支持代码高亮显示，行号显示，文本缩放，头文件和常用函数自动补全，快速注释/取消注释代码，代码自动缩进等功能。
simpleC的目的是给那些不方便使用PC的C/C++开发者一个简单的开发工具。
